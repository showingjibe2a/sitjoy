关于 SITJOY

SITJOY 是一个面向跨境电商企业的内部管理系统，专注于产品信息管理、Amazon广告数据管理以及订单产品SKU管理等核心业务流程。系统采用轻量级架构，部署在 Synology NAS 上，为团队提供高效、稳定的数据管理解决方案。

核心功能模块

  • 产品管理：
    • 货号管理 - 维护产品系列（SKU Family）及品类信息
    • 面料管理 - 管理面料代码、英文名称及图片资源
    • 材料管理 - 维护填充材料、框架材料及材料类型
    • 卖点管理 - 管理产品特性（Features）及关联品类
    • 认证管理 - 维护产品认证信息及相关文档
    • 品类管理 - 管理产品分类层级结构
  • 下单产品管理：管理工厂下单SKU，包含完整的规格、尺寸、重量、成本、包装信息，以及与面料、材料、卖点、认证的多对多关联关系
  • 销售产品管理：管理销售平台SKU，关联店铺/品牌、货号系列，自动从关联的下单SKU计算面料组合、规格名称，自动生成平台SKU（格式：货号-面料-规格，如 DC01-Brown-2B1C）
  • 店铺与品牌管理：维护电商平台店铺信息，关联销售平台类型与品牌
  • Amazon 广告管理：
    • 广告信息管理 - 维护广告活动（Campaign）、广告组（Ad Group）、产品组合（Portfolio）的三层级结构
    • 广告信息分类管理 - 管理广告信息细分类及关联的操作类型，操作类型区分活动/组适用范围
  • 图片管理：支持图片上传、预览、批量下载，采用网格布局展示

技术架构

  • 后端框架：Python WSGI 应用，使用 pymysql 进行数据库操作
  • 数据库：MySQL/MariaDB，采用 InnoDB 引擎，支持外键约束和事务
  • 前端技术：原生 HTML5 + CSS3 + JavaScript，无框架依赖
  • UI 设计：Morandi 色系（柔和的沙色、墨色、雾色、石板色），现代化卡片式布局
  • 部署环境：Synology NAS，轻量级部署，适合中小团队内部使用

数据库设计特点

  • 规范化设计：采用第三范式设计，减少数据冗余
  • 外键约束：使用 FOREIGN KEY 确保数据完整性和一致性
  • 多对多关系：通过中间表（Junction Table）实现复杂的多对多关联
  • 自动时间戳：created_at 和 updated_at 字段自动维护创建和更新时间
  • 迁移机制：支持数据库结构的增量更新和向后兼容

快速开始

  1. 在 Synology NAS 上安装 Python 3.8+ 环境
  2. 克隆项目代码到 NAS 目录
  3. 安装依赖：pip3 install -r requirements.txt
  4. 配置数据库：复制 db_config.example.json 为 db_config.json，填写数据库连接信息
  5. 启动应用：python3 app.py
  6. 访问系统：http://your-nas-ip:5000

主要 API 端点

  • GET/POST/PUT/DELETE /api/sku - 货号管理 API
  • GET/POST/PUT/DELETE /api/category - 品类管理 API
  • GET/POST/PUT/DELETE /api/fabric - 面料管理 API
  • GET/POST/PUT/DELETE /api/material - 材料管理 API
  • GET/POST/PUT/DELETE /api/feature - 卖点管理 API
  • GET/POST/PUT/DELETE /api/certification - 认证管理 API
  • GET/POST/PUT/DELETE /api/order-product - 下单产品管理 API
  • GET/POST/PUT/DELETE /api/sales-product - 销售产品管理 API
  • GET/POST/PUT/DELETE /api/shop - 店铺/品牌管理 API
  • GET/POST/PUT/DELETE /api/amazon-ad - Amazon广告信息 API
  • GET/POST/PUT/DELETE /api/amazon-ad-subtype - 广告信息分类 API
  • GET/POST/PUT/DELETE /api/amazon-ad-operation-type - 广告操作类型 API
  • POST /api/gallery/upload - 图片上传 API
  • GET /api/gallery/list - 图片列表 API
