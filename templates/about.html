📖 关于此项目

这是一个部署在 Synology NAS 上的 Python Web 应用项目。

技术栈

  • 后端框架： Python Flask
  • 前端： HTML5 + CSS3 + JavaScript
  • 部署环境： Synology NAS

项目结构


sitjoy/
├── app.py                 # Flask 主应用
├── requirements.txt       # 依赖文件
├── templates/             # HTML 模板
│   ├── index.html        # 首页
│   └── about.html        # 关于页面
└── static/               # 静态文件
    └── css/
        └── style.css     # 样式表

快速开始

  1. 在 NAS 上安装 Python 3.8+
  2. 安装依赖：pip install -r requirements.txt
  3. 运行应用：python app.py
  4. 访问：http://nas-ip:5000

API 端点

  • GET / - 首页
  • GET /about - 关于页面
  • POST /api/hello - 问候 API
  • GET /status - 系统状态
